­Ъћ╣ рЃЦрЃЏрЃюрЃўрЃА рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃа рЃАрЃбрЃўрЃџрЃА

­Ъћ╣ рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃцрЃћрЃарЃћрЃЉрЃА, рЃерЃарЃўрЃцрЃбрЃћрЃЉрЃА, рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА

­Ъћ╣ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃћрЃЉрЃА рЃерЃћрЃЦрЃЏрЃюрЃљрЃю рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃљрЃарЃњрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃќрЃћ

­Ъћ╣ рЃљрЃарЃўрЃА HTML-рЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃЌрЃў рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃА рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃўрЃА рЃњрЃљрЃарЃћрЃњрЃюрЃЮрЃЉрЃљрЃА

JavaScript

JavaScript рЃў рЃљрЃарЃўрЃА рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃћрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃЮрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ. рЃарЃЮрЃфрЃљ рЃерЃћрЃю рЃ«рЃљрЃа рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃЊрЃљ рЃ«рЃћрЃЊрЃљрЃЋ рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃа рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃАрЃџрЃљрЃўрЃЊрЃћрЃарЃћрЃЉрЃў, рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў рЃљрЃю рЃЌрЃљрЃЏрЃљрЃерЃћрЃЉрЃў Рђћ рЃћрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў JavaScript-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃљрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў

­Ъћ╣ рЃЦрЃЏрЃюрЃўрЃА рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃћрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃЮрЃЉрЃљрЃА

­Ъћ╣ рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃа рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃАрЃџрЃљрЃўрЃЊрЃћрЃарЃћрЃЉрЃў, рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў рЃЊрЃљ рЃЌрЃљрЃЏрЃљрЃерЃћрЃЉрЃў

­Ъћ╣ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃћрЃЉрЃА рЃерЃћрЃЦрЃЏрЃюрЃљрЃю рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃљрЃЊрЃў рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃћрЃю рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃќрЃћ

­Ъћ╣ рЃљрЃарЃўрЃА HTML-рЃўрЃА рЃЊрЃљ CSS-рЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃЌрЃў рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃФрЃџрЃўрЃћрЃарЃћрЃЉрЃА рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃЮрЃЉрЃљрЃА

Python

Python рЃў рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃћрЃЉрЃўрЃА, рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ. рЃарЃЮрЃфрЃљ рЃерЃћрЃю рЃ«рЃљрЃа рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃќрЃћ рЃЊрЃљ рЃ«рЃћрЃЊрЃљрЃЋ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃА, рЃљрЃю рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА Рђћ рЃћрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў Python-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃљрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў

­Ъћ╣ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃћрЃЉрЃўрЃА, рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ

­Ъћ╣ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў, рЃарЃљрЃф рЃЏрЃљрЃА рЃерЃћрЃАрЃљрЃюрЃўрЃерЃюрЃљрЃЋ рЃљрЃарЃЕрЃћрЃЋрЃљрЃюрЃА рЃ«рЃЊрЃўрЃА рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃЌрЃљрЃЌрЃЋрЃўрЃА

­Ъћ╣ рЃцрЃљрЃарЃЌрЃЮ рЃћрЃЎрЃЮрЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЊрЃљ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃФрЃџрЃўрЃћрЃарЃћрЃЉрЃћрЃю рЃЏрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃА

­Ъћ╣ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃћрЃфрЃюрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў, рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃБрЃа рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃерЃў рЃЊрЃљ рЃЋрЃћрЃЉрЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃерЃў

MMA

Goal Oriental Academy рЃљрЃа рЃерЃћрЃЏрЃЮрЃўрЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃЌ Рђћ рЃЕрЃЋрЃћрЃю рЃњрЃЋрЃ»рЃћрЃарЃљ, рЃарЃЮрЃЏ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃњрЃЮрЃюрЃћрЃЉрЃарЃўрЃЋрЃў, рЃўрЃАрЃћ рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ. рЃАрЃгрЃЮрЃарЃћрЃЊ рЃљрЃЏрЃўрЃбрЃЮрЃЏ, рЃљрЃЎрЃљрЃЊрЃћрЃЏрЃўрЃљрЃерЃў рЃЊрЃљрЃњрЃћрЃњрЃЏрЃўрЃџрЃўрЃљ MMA (Mixed Martial Arts) рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃАрЃљрЃЉрЃарЃФрЃЮрЃџрЃЮ рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃћрЃЉрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃ«рЃћрЃЮрЃЉрЃљрЃА, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃЉрЃарЃљрЃќрЃўрЃџрЃўрЃБрЃарЃў рЃ»рЃўрЃБ-рЃ»рЃўрЃбрЃАрЃБ, рЃЎрЃљрЃарЃљрЃбрЃћ, рЃЎрЃўрЃЎрЃЉрЃЮрЃЦрЃАрЃў рЃЊрЃљ рЃўрЃБрЃЊрЃЮ.

­Ъћ╣ рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃњрЃљрЃЏрЃФрЃџрЃћрЃЮрЃЉрЃљ рЃЊрЃљ рЃЊрЃўрЃАрЃфрЃўрЃърЃџрЃўрЃюрЃљ Рђћ MMA рЃњрЃљрЃгрЃЋрЃарЃЌрЃюрЃўрЃА рЃљрЃарЃљрЃЏрЃљрЃарЃбрЃЮ рЃАрЃ«рЃћрЃБрЃџрЃА, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃњрЃЮрЃюрЃћрЃЉрЃљрЃАрЃљрЃф. рЃўрЃА рЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЏрЃЮрЃЌрЃЏрЃўрЃюрЃћрЃЉрЃўрЃА, рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃАрЃгрЃарЃљрЃцрЃў рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљрЃерЃў Рђћ рЃўрЃАрЃћрЃЋрЃћ, рЃарЃЮрЃњрЃЮрЃарЃф рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃўрЃА рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃњрЃљрЃЊрЃљрЃГрЃарЃљ.

­Ъћ╣ рЃЌрЃЋрЃўрЃЌрЃЊрЃљрЃ»рЃћрЃарЃћрЃЉрЃБрЃџрЃЮрЃЉрЃљ рЃЊрЃљ рЃЌрЃљрЃЋрЃЊрЃљрЃфрЃЋрЃљ Рђћ рЃЕрЃЋрЃћрЃюрЃА рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃА рЃЋрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃБрЃюрЃљрЃарЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃљ рЃљрЃарЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃАрЃўрЃЋрЃарЃфрЃћрЃерЃў, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃарЃћрЃљрЃџрЃБрЃа рЃфрЃ«рЃЮрЃЋрЃарЃћрЃЉрЃљрЃерЃўрЃф. рЃфрЃЮрЃЊрЃюрЃљ рЃЌрЃЋрЃўрЃЌрЃЊрЃљрЃфрЃЋрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃљрЃФрЃџрЃўрЃћрЃарЃћрЃЉрЃА рЃЌрЃљрЃЋрЃЊрЃљрЃ»рЃћрЃарЃћрЃЉрЃБрЃџрЃЮрЃЉрЃљрЃА рЃЊрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃњрЃарЃФрЃюрЃћрЃЉрЃљрЃА.

­Ъћ╣ рЃЉрЃљрЃџрЃљрЃюрЃАрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљрЃАрЃљ рЃЊрЃљ рЃАрЃърЃЮрЃарЃбрЃА рЃерЃЮрЃарЃўрЃА Рђћ рЃЊрЃдрЃћрЃЋрЃљрЃюрЃЊрЃћрЃџ рЃфрЃўрЃцрЃарЃБрЃџ рЃћрЃърЃЮрЃЦрЃљрЃерЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃЉрЃљрЃџрЃљрЃюрЃАрЃўрЃА рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃљ. MMA рЃ«рЃћрЃџрЃА рЃБрЃгрЃДрЃЮрЃЉрЃА рЃАрЃбрЃарЃћрЃАрЃўрЃА рЃЏрЃЮрЃ«рЃАрЃюрЃљрЃА, рЃћрЃюрЃћрЃарЃњрЃўрЃўрЃА рЃљрЃдрЃЊрЃњрЃћрЃюрЃљрЃА рЃЊрЃљ рЃАрЃљрЃћрЃарЃЌрЃЮ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃБрЃџрЃЮрЃЉрЃўрЃА рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃљрЃА.
